Scanner für Log-Dateien (WebSphere/Liberty & Co.), der personenbezogene und kritische Daten wie Passwörter, Keys, Tokens, IBANs und Versicherungsnummern per Regexp-, Feld- und String-Mustern findet.
```

**Englisch:**
```
Log file scanner that detects personal and sensitive data (passwords, keys, tokens, IBANs, insurance numbers, ...) via regexp, field and string patterns. WebSphere/Liberty, Tomcat, Spring Boot, JSON.

## Weitere Empfehlungen

- **Repo-Einstellungen → Features:** „Issues" aktiviert lassen (Templates liegen bereit).
- **Include in the home page:** Häkchen bei „Releases" und „Packages" nach Bedarf.
- **Später ein Release taggen:** `git tag v1.6.1 && git push origin v1.6.1`,
  dann auf GitHub unter „Releases" ein Release aus `changelog.md` erstellen.
